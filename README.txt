CONNECTIFY - Modular C++ Project

How to Compile and Run:
------------------------
1. Open a terminal or command prompt in this folder.
2. Compile using g++:
   g++ main.cpp auth.cpp profile.cpp chat.cpp posts.cpp notifications.cpp -o connectify
3. Run the program:
   ./connectify

Modules:
- auth.cpp/.h          : Handles login authentication
- profile.cpp/.h       : Profile management (name, bio)
- chat.cpp/.h          : Simple chat system
- posts.cpp/.h         : Post creation
- notifications.cpp/.h : Notification viewer
